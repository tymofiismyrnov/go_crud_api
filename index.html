Simple API in Go with Gin

This is a simple API application developed in Go using the Gin library.

PostgreSQL Routes:

  • GET /posts: Retrieve all posts

    • curl http://localhost:8080/posts

  • GET /posts/:id: Retrieve a post by ID

    • curl http://localhost:8080/posts/123

  • POST /posts: Create a new post

    • curl -X POST -H "Content-Type: application/json" -d '{"title":"New Post","content":"This is a new post."}' http://localhost:8080/posts

  • PUT /posts/:id: Update a post by ID

    • curl -X PUT -H "Content-Type: application/json" -d '{"title":"Updated Post","content":"This post has been updated."}' http://localhost:8080/posts/123

  • DELETE /posts/:id: Delete a post by ID

    • curl -X DELETE http://localhost:8080/posts/123

Redis Routes:

  • POST /redis: Set a key-value pair in Redis

    • curl -X POST -d "key=mykey&value=myvalue" http://localhost:8080/redis

  • GET /redis: Retrieve a value from Redis

    • curl http://localhost:8080/redis?key=mykey

  • DELETE /redis: Delete a key from Redis

    • curl -X DELETE http://localhost:8080/redis?key=mykey

Environment Variables:

Environment variables should be stored in the .env file since it's exported by the initializers.LoadEnvVariables() function.

  • PORT: The port on which the API is listening.
  • DB_URL: PostgreSQL connection string in the format:
    DB_URL="host=hostname user=username password=yourpass dbname=dbname port=port sslmode=disable"
  • REDIS_HOST: Redis host in the format: url:port
  • REDIS_PASSWORD: The password for Redis.
